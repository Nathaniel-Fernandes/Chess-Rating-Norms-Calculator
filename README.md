# USCF Rating & Norms Calculator

I started this project because I could not find a quick way to calculate whether or not I would earn a norm after playing in a chess tournament. Hopefully this helps you!

Access online [here](https://chessintellect.com/tech/uscf-rating-norm-calculator/) or paste [chessintellect.com/tech/uscf-rating-norm-calculator/](https://chessintellect.com/tech/uscf-rating-norm-calculator/) into your browser address bar.

## Installation Instructions
Clone the repository and run the following scripts to try it out for yourself!
```
npm install
npm run build
npm run serve
```
## Technologies Used
- Vue.js
- TypeScript
- HTML/CSS
